# 🏦 Bank Loan Analysis Dashboard (Power BI)

## 📌 Project Overview

This project is a Power BI dashboard that analyzes bank loan data to provide insights into loan distribution, repayment behavior, and risk assessment.

The dashboard helps financial institutions understand loan performance and make data-driven decisions.

---

## 🎯 Objectives

* Analyze total loan applications and funding trends
* Track repayment and received amounts
* Identify good and bad loans
* Monitor monthly loan performance
* Provide insights for decision-making

---

## 📊 Key Features

* Total Loan Applications KPI
* Total Funded Amount
* Total Amount Received
* Loan Status Analysis (Good vs Bad Loans)
* Monthly Trends Visualization
* Interactive Filters and Slicers

---

## 🛠️ Tools & Technologies

* Power BI
* Excel / CSV Dataset
* SQL (optional for data cleaning)

---

## 📁 Dataset

The dataset contains information about:

* Loan ID
* Issue Date
* Loan Amount
* Total Payment
* Interest Rate
* Loan Status
